# SIFT+ Installer Script

**Automate the Setup of Your Ultimate Forensics Environment with Ease**

## Overview

The SIFT+ installer script is designed to streamline the setup of an advanced forensic workstation on Ubuntu 20 or Ubuntu 22. This script leverages the CAST project for SIFT 
workstation saltstack configuration, ensuring a consistent and reproducible environment.

## Features

- **Network Miner**: Network analysis tool.
- **OLE Tools**: Malware analysis tools.
- **Plaso Tools**: For timeline generation and analysis.
- **Autopsy**: A powerful forensic investigation tool.
- **Git**: Version control system.
- **Nmap**: Network scanning and auditing tool.
- **Gimp**: Image editing software.
- **Shotwell**: Desktop background changer.
- **KeePassXC**: Secure password manager.
- **OpenSSH Server**: Enables secure remote access.
- **P7zip-full**: Compression and extraction utility.
- **Xtrlock**: Locks the screen.
- **Whois**: Queries WHOIS databases for domain name registration information.
- **eSpeak**: Text-to-speech conversion.
- **Remmina**: Remote desktop protocol client.
- **GParted**: Disk partitioning tool.
- **Chkrootkit**: Rootkits scanner.
- **Evolution with Evolution-ews**: Email client and MS Exchange Web Server Plugin.
- **GNOME-Todo**: Sticky notes/task tracker.
- **Aspell**: Spell checker for Evolution.
- **Fierce**: DNS reconnaissance tool.
- **Net-tools**: Network-related utilities (netstat, etc.).
- **Sysstat**: Performance monitoring tools.
- **Lynis**: Security auditing tool.
- **LibreOffice**: Office productivity suite.
- **GNOME-Tweaks**: GNOME customization options.
- **GNOME Shell Extensions**: Enhances the GNOME desktop environment.
- **SQLite Browser**: Database management tool.
- **Visual Studio Code (VSCode)**: Source code editor.
- **PyCharm Community Edition**: Python IDE.
- **ZAPROXY**: Web application security scanner.
- **CherryTree Notes**: Hierarchical note-taking application.
- **VLC Media Player**: Multimedia player.
- **Cmatrix**: Terminal animation for fun and distraction.
- **CowSay**: Fun terminal utility to output text as speech bubbles.
- **Sl**: Replacement for ls with ASCII art (for misstyping 'ls').
- **Cheese**: Webcam photo booth.

## Installation

To get started, simply execute the following commands:

```bash
chmod +x ./installSIFTplus.sh && ./installSIFTplus.sh
```
Watch for several prompts asking for your sudo password to install certain packages.
